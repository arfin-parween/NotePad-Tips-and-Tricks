# NotePad-Tips-and-Tricks
Notepad++ is a free source code editor. This Repository Contains all the basic settings and tips related to notepad++.


## Hey There!

Notepad++ is a popular text editor and source code editor for Windows. It supports several programming languages and offers features like syntax highlighting, code folding, auto-completion, and macro recording. Notepad++ is known for its lightweight nature and fast performance, making it a favorite among developers for writing and editing code. It also supports plugins, allowing users to extend its functionality further.
